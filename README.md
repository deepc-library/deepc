# DeepC

A lightweighted deep learning library implemented in C.

## Quick Install

```bash
git clone https://github.com/Abdul-Haseeb-0313/DeepC.git
```

## Compile test with:
```bash
mkdir build 
cd build 
cmake ..
cmake --build .
cd ..
bin/test
```

## Features
- Neural networks with multiple layer types

- Matrix operations optimized for performance

- Model saving/loading in .dc format

- Data preprocessing and CSV loading

- No external dependencies
